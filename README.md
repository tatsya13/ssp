# 🎯 Интерактивное задание: Карта ССП

Интерактивное образовательное веб-приложение для изучения Сбалансированной системы показателей (ССП/Balanced Scorecard).

## 📖 Описание

Приложение предназначено для изучения темы 4 дисциплины «Управление физкультурно-спортивными организациями». Пользователям предлагается распределить 18 карточек с показателями эффективности по четырём блокам ССП.

## 🏗️ Структура ССП

Приложение охватывает четыре перспективы Сбалансированной системы показателей:

| Перспектива | Количество карточек | Описание |
|-------------|---------------------|----------|
| 📚 **Обучение и развитие** | 5 | Квалификация кадров, технологии, вложения в развитие, инновационность |
| ⚙️ **Внутренние процессы** | 6 | Эффективность процессов, качество тренировок, инновации, отчётность |
| 👥 **Клиенты / Спортсмены** | 5 | Удовлетворённость, удержание, доля рынка, лояльность |
| 💰 **Финансы** | 3 | Рентабельность, внебюджетные доходы, бюджет, объёмные показатели |

## 🚀 Использование

1. Откройте `index.html` в любом современном браузере
2. Перетащите карточки из банка показателей в соответствующие блоки ССП
3. Нажмите **«Проверить»** для оценки результата
4. При необходимости используйте кнопку **«Показать правильный вариант»** для самопроверки

## ✨ Возможности

- **Drag-and-drop интерфейс** — перетаскивание карточек мышью
- **Поддержка сенсорных экранов** — работа на мобильных устройствах
- **Мгновенная проверка** — подсчёт правильных ответов с обратной связью
- **Визуализация правильного ответа** — модальное окно с эталонным распределением
- **Адаптивный дизайн** — корректное отображение на разных экранах
- **Печать** — возможность распечатать результат

## 🎓 Логика ССП

Приложение демонстрирует причинно-следственную связь между перспективами:

> Инвестиции в обучение персонала → Совершенствование процессов → Рост удовлетворённости клиентов → Увеличение доходов → Реинвестирование в кадры

## 📝 Показатели

В приложении представлены 18 показателей эффективности, включая:

- Индекс удовлетворённости спортсменов и родителей
- Доля внебюджетных доходов
- Количество освоенных компетенций
- Процент удержания спортсменов
- Соблюдение графика подготовки мероприятий
- И другие метрики
